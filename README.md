# ⌨️ Typing Speed Test Game

A simple and interactive **Typing Speed Test** web application built using **JavaScript**, **HTML**, and **CSS**. This project helps users measure their typing speed and accuracy in real-time.

---

## 🚀 Features
- ⏱️ Real-time typing speed calculation (WPM)
- 🎯 Accuracy tracking
- 📝 Random text generation
- 🔄 Restart functionality
- 📊 Instant results
- 💻 Responsive UI

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
